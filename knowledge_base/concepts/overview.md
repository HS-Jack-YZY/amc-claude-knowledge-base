# AMC Overview & Availability

## What is AMC?

Amazon Marketing Cloud (AMC) is a **privacy-safe, cloud-based clean room** solution from Amazon Ads. It enables advertisers to perform analytics across pseudonymized, event-level signals — including Amazon Ads signals along with their own inputs.

Key characteristics:
- **Clean room environment** — data never leaves the secure environment; only aggregated, anonymized results are exported
- **Privacy-safe** — all data is pseudonymized; raw user-level data cannot be exported
- **Cloud-based** — no infrastructure management required; Amazon hosts and manages the environment
- **SQL-based** — analysts write SQL queries (AMC SQL) to explore and aggregate data

## Benefits for Advertisers

AMC supports a wide range of advertising analytics use cases:

1. **Campaign Insights** — Measure reach, frequency, and performance across Amazon DSP and Sponsored Ads
2. **Customer Journey Analysis** — Understand the path users take from ad exposure to conversion
3. **Media Mix Optimization** — Evaluate the relative impact of different ad channels and formats
4. **Audience Segment Analysis** — Analyze performance across audience segments and demographics
5. **Path-to-Conversion** — Map the sequence of touchpoints leading to purchases
6. **Audience Creation** — Build custom audiences for activation in Amazon DSP campaigns
7. **Paid Features Analytics** — Access premium datasets (retail purchases, brand store insights, etc.)
8. **Cross-Channel Attribution** — Combine Amazon signals with advertiser-uploaded first-party data

## Key Features

- **OAuth 2.0 authentication** — secure API access using standard OAuth flows
- **Multi-user access** — multiple analysts can work within the same AMC instance
- **Functional API categories** — separate APIs for instance management, workflow management, audience management, and reporting
- **Web UI + API** — accessible through both the AMC web interface and the Amazon Ads API
